Django Powered Backend WebService
-

An opinionated starter-template django powered backend api webservice with simplicity in mind.

#### pre-commit code quality hooks
- isort and isort-config
- black
- flake8

#### git commit message style
    new:    new business logic feature
    dev:    new software feature
    inf:    infrastructures, deployment
    fix:    bug fix
    ref:    refactoring production code
    doc:    add/update documentation
    test:   adding/refactoring test; no production code change
    style:  formatting, missing semi colons, typos, etc; no code change
    chore:  updating build tasks, package manager configs, etc; no production code change
